# linux基础过

### 1.文件系统与命令

- `pwd`：打印目前的工作目录

- `ls`：打印出当前文件夹的文件

- `cd`：进入目录

- `mkdir`：创建目录

- `touch`：创建文件

- `cp`：复制文件

- `mv`：移动文件

- `rm`: 删除文件 -rf：r:递归 f：force强制

- `cat`：查看文件内容，可加>把新的内容加进去

  1. cat > 文件名

  2. cat << EOF > 文件名
     内容行1
     内容行2
     EOF

  3. cat <<< "内容" > 文件名

  4. cat 源文件 > 目标文件

  5. echo "内容" | cat > 文件名

     >- `cat > 文件` → **从头覆盖**文件，旧内容全部丢失。
     >- `cat >> 文件` → **追加**到文件末尾，旧内容完整保留。

- `less`：cat的少字版，需要翻页;上下箭头移动一行，空格键翻页；大小写G一个开头一个末尾

- `head`：和head相反，看前十行的内容

- `tail`：看文件最后十行内容（默认），参数tail -n 制定行数 文件名可以自行更正行数；tail -f可跟随，实时打印行数

- `grep`：查找，用法：grep [查询的关键字] 文件名（可多文件，可写通配符），grep -i [查询关键字] 文件名（忽略大小写） -P正则表达式匹配

- `find`：查找文件命令，常见用法 find .-name "文件名"； find .-iname "文件名"（忽略大小写）；find .-type/ f代表普通文件；d代表文件

- 了解EOF终止符

- ctrl c强行终止；ctrl z暂停终止

- `sed`:流编辑器（Stream Editor）

  Sed [选项] '地址 动作' 文件名

  1. 常用选项：-n：“只打印匹配行”；-i：“直接修改原文内容”； -e：允许执行多个编辑操作

  2. 地址：数字：n（第n行）、1，5（第一到第五行）、$最后一行；正则匹配：/pattern/（包含该关键字的行）

  3. 动作：p（打印print）、d（delete删除）、s（替换subsititute）

     >你的电脑使用的是 **macOS (Mac) 系统**。Mac 自带的 `sed` 工具是 BSD 版本，它的 `-i` 参数语法和 Linux 有点区别。Mac 的 `-i` 后面**强制要求**紧跟一个备份文件的后缀名（如果是空字符串，必须写成 `''`）。

  4. `awk`文本分析工具：

     awk  [选项]  '模式{动作}' 文件名

     1. 常用选项： -F：指定列的分隔符
     2. 内置变量：$0：当前整行内容；$1,$2. $n :代表当前行的第1，第2第n列；NR：当前处理的行号；NF：当前处理的列号

### 3.进程与日志

>```

>top：实时动态管理器
>kill：杀掉进程，用法：kill pid
>tail -f：看末尾的日志
>```
>
>stdout / stderr/stdin：标准输出报错输入，分别是120 &全部存盘
